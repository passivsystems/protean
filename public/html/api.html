API

Admin

Curl command Details
curl -v http://localhost:3001/services Get a list of services
curl -v http://localhost:3001/services/sample Get the sample codex (service definitions)
curl -v http://localhost:3001/services/sample/usage Get the sample usage (curl command form)
curl -v -X POST -H "Content-Type: application/json" --data '{"locs":["sample"]}}' 'http://localhost:3001/test' Integration test the service simulation
curl -v -X PUT http://localhost:3001/services --data-binary "@/path/sample.edn" Put a codex (service definitions) into Protean
curl -v -X DELETE http://localhost:3001/services/sample Delete a service from Protean
curl -v -X DELETE http://localhost:3001/services/sample/errors Turn off simulated response errors for this service
curl -v -X PUT http://localhost:3001/services/sample/errors/status/500 Add a simulated 500 error response for this service
curl -v -X PUT http://localhost:3001/services/sample/errors/probability/20 Set the probability of a simulated error to 20% for this service
